Proyecto: Damas

Contenido: Este proyecto implementa el clásico juego que todos conocemos llamado damas internacionales en c++. Permite que dos jugadores participen alterando turnos, realizando movimientos válidos, capturas y las coronaciones según las reglas tradicionales del juego.

Caracteristicas principales del juego:
- Tablero 10x10 visualizado en consola.
- Sistema de turnos entre jugadores.
- Validación de movimientos y capturas.
- Tiene detección automática de coronaciones y las condiciones de victoria.
